# Chrome Web Store Listing (Copy/Paste)

## Extension name
Zenovee Tools

## Short description (up to ~132 chars)
Access Zenovee paid tool categories from your browser. Login to unlock tools based on your active subscription.

## Detailed description
Zenovee Tools is the official Chrome extension for Zenovee (zenovee.in).

### What it does
- Lets you sign in with your Zenovee account
- Shows your purchased tool-category access (entitlements)
- Provides quick links to Dashboard and Tools

### Paid-only access
Zenovee is a paid-only tool platform. Installing the extension does not grant free access.
Tools unlock only for users with an active subscription/entitlement.

### Permissions justification
- Storage: used to store your login session securely in the extension so you don’t need to login every time.
- Host permissions (zenovee.in): used to call Zenovee API endpoints to fetch your entitlements.
